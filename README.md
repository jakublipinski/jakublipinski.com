# jakublipinski.com
jakublipinski.com website

# Installation (MacOS)

```
gem install --user-install bundler
sudo gem install -n /usr/local/bin/ jekyll
```

# Run
```
export PATH=~/.gem/ruby/2.6.0/bin:$PATH
bundle exec jekyll serve --livereload
```
